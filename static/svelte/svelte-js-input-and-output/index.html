Welcome to your introductory guide to SvelteKit! If you already have a grasp of basic HTML, CSS, and JavaScript, you're well on your way to mastering SvelteKit. In this guide, we'll start with one of the most fundamental concepts in SvelteKit: outputting variables into HTML.

In SvelteKit, you can easily insert the value of a JavaScript variable into your HTML using curly brackets {}. Let's go through a simple example to see how this works.

<script>
  let name = 'world'
</script>

<h1>Hello {name}!</h1>

Explanation

Script Tag: The script tag is used to define JavaScript variables and functions that you want to use in your component. Here, we declare a variable name and assign it the value 'world'.

<script>
  let name = 'world'
</script>

Curly Brackets: To output the value of the name variable in your HTML, you simply wrap the variable in curly brackets {}. This line will show as an HTML h1 element with the text content Hello world!.

<h1>Hello {name}!</h1>

When the app runs, it processes the code, and wherever it finds {name} in the HTML, it replaces it with the value of the name variable. In this example, it changes Hello {name}! to Hello world!.

Try It Yourself

  1. Open your SvelteKit project.
  2. Add the example code above to one of your Svelte files (e.g., src/routes/+page.svelte).
  3. Run your SvelteKit project and open it in your browser to see the result.

In addition to outputting variables into text content, SvelteKit allows you to use variables for HTML attributes, such as the src attribute of img elements. This is very useful for dynamically setting attributes based on your JavaScript code.

<script>
  let imageUrl = 'https://picsum.photos/200'
</script>

<img src={imageUrl} alt="Example image">

Explanation

Script Tag: Just like before, we start by defining our JavaScript variable within a <script> tag. Here, imageUrl is a variable holding the URL of an image.

<script>
  let imageUrl = 'https://picsum.photos/200'
</script>

Curly Brackets in Attributes: To set the src attribute of the img element using the imageUrl variable, we place the variable inside curly brackets {} within the attribute. This tells SvelteKit to evaluate the imageUrl variable and use its value as the src attribute of the img element.

<img src={imageUrl} alt="Example image">

When your app runs, wherever it finds {imageUrl} in the src attribute, it replaces it with the value of the imageUrl variable. In this example, it changes img src={imageUrl} alt="Example image" to img src="https://picsum.photos/200" alt="Example image".

You can easily bind HTML inputs to JavaScript variables to get better interactions and real-time updates in your app. This is particularly useful for creating interactive user interfaces. In this section, we will see how to use an input element to assign a value to a variable.

SvelteKit provides a feature called binding, which allows an input field to display a variable's value and update the variable when the user types into the input. This is done using bind:value.

<script>
  let name = 'world'
</script>

<input bind:value={name} placeholder="Enter your name">

<h1>Hello {name}!</h1>

Explanation

Script Tag: As before, we define a variable within the script tag.

<script>
  let name = 'world'
</script>

Input Element with Binding: The input element uses bind:value to connect the value attribute of the input to the name variable. When the user types in the input field, the name variable is automatically updated with the new value.

<input bind:value={name} placeholder="Enter your name">

Using the Variable in HTML: The value of the name variable is then used in the HTML, just as before. This line will update to show the value of the name variable as the user types.

<h1>Hello {name}!</h1>

Try It Yourself

  1. Open your SvelteKit project.
  2. Add the example code above to one of your Svelte files (e.g., src/routes/+page.svelte).
  3. Run your SvelteKit project and open it in your browser to see the result.

You can do this for other types of input elements such as checkboxes, radio buttons, and select dropdowns. Here are a few more examples:

Example with a Checkbox

<script>
  let isChecked = false
</script>

<input type="checkbox" bind:checked={isChecked}>

<p>The checkbox is {isChecked}.</p>

Example with a Select Dropdown

<script>
  let selectedOption = 'Option 1'
</script>

<select bind:value={selectedOption}>
  <option value="Option 1">Option 1</option>
  <option value="Option 2">Option 2</option>
  <option value="Option 3">Option 3</option>
</select>

<p>You selected: {selectedOption}</p>

You can use a button to update the value of a variable by saying what function to run when it's clicked. This allows you to perform actions, such as updating variables.

Here's a simple example where a button click updates a variable value:

<script>
  let count = 0

  function increment() {
    count = count + 1
  }
</script>

<p>Count: {count}</p>
<button on:click={increment}>Increment</button>

Explanation

Script Tag: The variable and the function are defined inside the script tag. count is a variable initialised to 0 and increment is a function that increases the value of count by 1.

<script>
  let count = 0

  function increment() {
    count = count + 1
  }
</script>

Display the Variable: Curly brackets {} display the count variable in the HTML.

<p>Count: {count}</p>

Button connected to a function: on:click={increment} connects the increment function to the button's click event. When the button is clicked, the increment function is called, updating the count variable.

<button on:click={increment}>Increment</button>

Try It Yourself

  1. Open your SvelteKit project.
  2. Add the example code above to one of your Svelte files (e.g., src/routes/+page.svelte).
  3. Run your SvelteKit project and open it in your browser to see the result.

Updating Multiple Variables

You can use buttons to update multiple variables or trigger more complex functions. Here's an example where two buttons increase and decrease a count variable:

<script>
  let count = 0

  function increment() {
    count = count + 1
  }

  function decrement() {
    count = count - 1
  }
</script>

<p>Count: {count}</p>
<button on:click={increment}>Increment</button>
<button on:click={decrement}>Decrement</button>
