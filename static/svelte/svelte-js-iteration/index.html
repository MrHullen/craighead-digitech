Using {#each} Statements in SvelteKit

In Svelte, {#each} statements are used to iterate over arrays and render HTML for each item. This is similar to using forEach loops in vanilla JavaScript, but it integrates directly with Svelte's templating system, making it easier to create dynamic lists.

Before we get into {#each} statements in Svelte, let's quickly recap how forEach loops work in vanilla JavaScript.

const students = [
  'Alice',
  'Barbie',
  'Charlie'
]

students.forEach(student => {
  alert(student)
})

In this example, the forEach loop iterates over the students array and outputs each student's name.

In Svelte, you can use the {#each} block to iterate over an array and show content for each item in there. Here's an example where we loop through an array of student names and display them in an unordered list.

<script>
  let students = [
    'Alice',
    'Barbie',
    'Charlie'
  ]
</script>

<ul>
  {#each students as student}
    <li>{student}</li>
  {/each}
</ul>

Explanation

Script Tag: Define the array inside the script tag.

<script>
  let students = [
    'Alice',
    'Barbie',
    'Charlie'
  ]
</script>

Iterating with

{#each}: Use the {#each} block to iterate over the students array and render a list item for each student.

<ul>
  {#each students as student}
    <li>{student}</li>
  {/each}
</ul>
  • {#each students as student}: This block iterates over the students array and assigns each item to the student variable.
  • li {student} /li : For each item in the array, this line renders a list item (li) containing the student's name.
  • {/each}: This closes the each block.

Try It Yourself

  1. Open your SvelteKit project.
  2. Add the example code above to one of your Svelte files (e.g., src/routes/iteration/+page.svelte).
  3. Try changing the values in the students array and adding more value and see what happens.
  4. Run your SvelteKit project and open it in your browser to see the result.

You can also use the index of each item in the array by including index after as in the {#each} block.

<script>
  let students = [
    'Alice',
    'Barbie',
    'Charlie'
  ]
</script>

<ul>
  {#each students as student, index}
    <li>{index + 1}. {student}</li>
  {/each}
</ul>

Here, the index variable provides the position of each item in the array, starting from 0.

In this exercise, you will practise using {#each} blocks to iterate over an array of objects and show a list of products. Each product will have a name and a price, which will be displayed in HTML.

  1. Create a New Svelte File: Create a new Svelte file (e.g., ProductList.svelte) in your lib/ folder..
  2. Define the Product Array: In the script tag, define an array of product objects. Each object should have a name and a price property.
  3. Use the {#each} Block: Use the {#each} block to iterate over the product array and show each product's name and price in an unordered list (ul).

Here's what your ProductList.svelte script section might look like:

<script>
  let products = [
    { name: 'Apple', price: 1.0 },
    { name: 'Banana', price: 0.5 },
    { name: 'Cherry', price: 2.0 },
  ]
</script>

Remember you can show values inside an object by using dot notation like product.name or product.price.

Tip: You can show the price to two decimal places by adding .toFixed(2) inside the curly brackets, {product.price.toFixed(2)} (source).

Extension

  1. Add a Description: Extend each product object to include a description property and display this information below the product's name and price.

Mandatory Extra

  1. Make it a component: Take the code you've written for an individual item and put it in a new component, e.g. ProductItem.svelte. Replace the object references with exported variables, and send each product's information in the {#each} block to the component as props.
