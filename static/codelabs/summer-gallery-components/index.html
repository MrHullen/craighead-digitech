Open VS Code

  1. Once it has cloned to your computer, click the Open in Visual Studio Code button (or press Command + Shift + A on MacOS or Ctrl + Shift + A on Windows)
  2. Open the file called index.svelte in the src folder

Start your dev server

  1. Open a new Terminal by clicking New Terminal in the Terminal menu, or pressing Ctrl + Shift + `
  2. Install all the dependencies by entering the command npm install
  3. Start the server on your laptop by entering the command npm run dev
  4. Open the link that it creates to view your site

The server will update the site in your browser every time you save a file, so that you can easily see the changes as you make them.

Currently, the header HTML and CSS is all mixed in with the rest of the page. This makes it difficult if you wanted to update the header specifically, or you wanted to re-use it on other pages.

We're going to put all of the code for the header into its own file, called a component. Then, we can add this component to the page without seeing all of its details. This will make our page much easier to understand, update, improve, and maintain.

Create the component

  1. Create a new file in your lib folder called Header.svelte
  2. Copy and paste the header HTML and CSS from index.svelte to Header.svelte

Use the component

  1. Add the script code below to the top of your index.svelte file
<script>
  import Header from '$lib/Header.svelte'
</script>
  1. Add the
    component into your HTML where you want the header to be

Components

A web app is made up of one or more components. A component is a reusable, self-contained block of code that includes HTML, CSS, and JavaScript that belong together, written into one file. That file can then be included with other components to create the page.

New vocabulary: component

Commit point

Now that you've added a feature to your website, you should commit it to your GitHub repository! Committing creates a full backup and a snapshot of the code at this point.

Open GitHub Desktop and check that the Current Repository in the top left is your project, and briefly review the changes you've made. Green indicates code that you've added since your last commit, and red is code you've deleted.

When you're happy, write a short (2 - 5 words) Summary of what you added in the bottom left. In this case, your message could be Created header component.
